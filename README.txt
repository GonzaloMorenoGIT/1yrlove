MATCH LOVE
==========

Página web de cuenta atrás para el aniversario de Gonzalo y Almudena.

FECHA CONFIGURADA
-----------------
17 de julio de 2026 a las 15:00, hora de Madrid.

CÓMO ABRIRLA
------------
1. Descomprime el archivo ZIP.
2. Abre el archivo index.html con Safari, Chrome o cualquier navegador.

CÓMO PUBLICARLA GRATIS
----------------------
Puedes subir la carpeta a:
- GitHub Pages
- Netlify
- Cloudflare Pages

MÚSICA OPCIONAL
---------------
Para añadir música:
1. Copia un archivo MP3 dentro de la carpeta y llámalo musica.mp3.
2. En index.html, busca:
   <source src="" type="audio/mpeg">
3. Sustitúyelo por:
   <source src="musica.mp3" type="audio/mpeg">
